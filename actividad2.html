A continuación encontraremos un ejercicio práctico para la construcción de un archivo HTML simple, realizar paso a paso.


1. Vamos a crear en un Bloc de Notas un archivo de texto, vacío por ahora, llamado metadato.txt en su equipo.


2.Copie las caracteristicas de Los metadatos y la World Wide Web desde las lecturas vistas al portapapeles, abra el archivo de texto metadato.txt con Bloc de Notas, pegue el contenido indicado y guárdelo.


Lo anterior con el fin de hallar la diferencia en la extención a la hora de crear el archivo a visualizar en el navegador*


3.Con Bloc de Notas, cree un archivo HTML mínimo y llámelo metadato.html


4.Copie nuevamente el texto de la World Wide Web , insértelo entre las indicaciones y del archivo metadato.html y guárdelo.


5.Abra metadato.html con Internet Explorer, Firefox u otro navegador para ver el resultado.



El siguiente ejercicio tiene como fin enseñar la herramienta de generación de metadatos.


Agregar los Metadatos Dublin Core con ayuda de una planilla (template) o generador de metadatos


1. Vaya a la página http://www.webposible.com/utilidades/dublincore-metadata-gen


2. Llene todos los campos que estime adecuados y necesarios para la correcta identificación del archivo metadato.html (creado en el ejercicio anterior)


3. Cuando haya terminado, clique el botón "Generar Metadatos"


4. Verá una lista con los metadatos solicitados, correctamente formateados para un documento HTML simple.


5. Copie toda la lista de metadatos al portapapeles.


6. Abra metadato.html con Bloc de Notas, posicione el cursor entre <head> y </head> y pegue los metadatos. Guarde el archivo. (para hacer mas fácil la edición es aconsejable que utilicemos editor de codigo como Sublime Text y así puedas identificar las etiquetas nombradas)


7. Abra metadato.html con Internet Explorer o cualquier otro navegador y examine los resultados. Examine igualmente el código fuente. (¿No sabe cómo? utiliza las teclas Control + U)
