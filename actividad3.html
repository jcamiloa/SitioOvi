El siguiente ejercicio tiene como fín entender básicamente la creación de un archivo html y su inclusión de metadatos, e inclusive identificar las partes del mismo.


Teniendo en cuenta que para la creación de un archivo HTML mínimo debe contener los siguientes elementos HTML mínimos:


<html> Abre el archivo HTML

<head> Abre la "cabeza" del archivo

<title> Aquí va el título que aparecerá en la barra azul del navegador </title>


</head> Cierra la "cabeza"

<body> Abre el "cuerpo"

Aquí debe ir el texto del archivo

</body> Cierra el "cuerpo"

</html> Cierra el archivo HTML
